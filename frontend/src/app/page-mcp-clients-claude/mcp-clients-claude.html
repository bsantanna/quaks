MCP Client

Claude

Connect your personal agent to Claude via the Quaks MCP plugin.

How to use with Claude

  1. Install the Quaks Claude plugin from the marketplace: /plugin install quaks-agents@quaks
  2. Authenticate in Claude: /quaks-ai:authenticate then complete the flow with /quaks-ai:complete-authentication.
  3. Your personal agent is now reachable via Claude's MCP tools and prompts.
